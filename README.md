# Premium Developer Portfolio

A high-performance, modern developer portfolio meticulously engineered to reflect a **Founder • Systems Engineer • SaaS Architect** brand identity.


## 🚀 Mission
To showcase professional engineering capabilities through technical storytelling, interaction design, and robust product architecture.

## 🛠 Modern Technical Stack

### Core Frameworks
- **Svelte 5 (Runes)**: Leveraged the latest reactive primitives for an ultra-responsive UI.
- **SvelteKit**: High-performance meta-framework for server-side rendering and routing.
- **Tailwind CSS**: Utility-first CSS for a precise, maintainable design system.

### Interaction & UX
- **GSAP & ScrollTrigger**: Powering "Liquid UX" with frame-accurate, scroll-synchronized animations.
- **Lenis**: Integrated for silky-smooth virtual scrolling across all desktop and mobile browsers.
- **Lucide Svelte**: Industry-standard symbolic iconography for high information density.

### Backend & Infrastructure
- **Supabase**: Real-time message persistence and secure data management.
- **Nodemailer (Gmail SMTP)**: Automated, branded email notification and confirmation system.
- **System Theme Sync**: 3-state theme engine (Light/Dark/System) with real-time detection.

## ✨ Key Features

### 1. Liquid UX Engine
A unified animation system where scroll position, entrance reveals, and magnetic micro-interactions are synchronized into a single, cohesive experience.

### 2. Specialized Product Highlight (Fairlx)
A dedicated showcase for Fairlx, featuring live interaction through theme-synchronized iframe previews and standard-compliant case study links.

### 3. Dynamic Technical Arsenal
A comprehensive skills matrix categorized by expertise (Frontend, Backend, Cloud, etc.) with over 30+ technology-specific icons and interactive tilt effects.

### 4. Professional Communication Flow
A custom-built contact form with server-side validation, Supabase integration, and automated SaaS-style email responses.

## 🏁 Getting Started

### Development
```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

### Production
```bash
# Build the project
npm run build

# Preview production build
npm run preview
```
